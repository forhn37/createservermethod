Node.js의 지원 모듈 중에서, 단연 사용빈도가 높은 http 모듈은 HTTP 프로토콜이 요구하는 갖가지 요건들을 꽤 높은 수준(편리한)으로 지원하고 있습니다. require 하는것 만으로도 너무나 쉽게 HTTP 프로토콜의 수백가지 요건들을 충족시킬 수 있게 되었다는 것이 다른언어와의 큰 차이입니다.
HTTP 프로토콜과 Node.js의 http모듈을 구분해서 Node.js에서의 http 모듈이 지원하는 메서드(객체의 속성으로 함수가 들어있는)중에 핵심 메서드인 createServer()는 요청과 응답을 처리하는 특수한 방식을 편리하게 사용할 수 있도록 기능(함수)을 지원하는데, 매개변수의 작성 조건이 콜백함수, 즉 '호출 할 때 개발자가 원하는 로직을 결정' 하는 방식을 채택하였습니다.

여기에는 통신의 주요 행동이라 할 수 있는 요청과 응답을 각각 객체에 담아 개발자가 편히 사용하도록 전달해주고 있습니다.
